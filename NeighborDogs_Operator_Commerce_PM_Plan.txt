================================================================================
DEFINITION OF DONE (the goal)
================================================================================
An operator onboards, sets walk prices in NeighborDogs, and a customer books and pays
the operator's price. The charge lands on the operator's Stripe account. NeighborDogs
receives $0 and carries no liability. The operator runs daily commerce (pricing, viewing
payments, refunds) entirely inside NeighborDogs. Card/bank data lives only in Stripe.

The project is DONE when: a real walk is paid end-to-end, operator is paid, NeighborDogs
gets $0, the webhook marks the booking paid, AND the operator can set prices, see payments,
and issue a refund without leaving NeighborDogs.
